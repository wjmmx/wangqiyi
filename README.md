# 结庐

> "结庐在人境，而无车马喧"

收集了叔父王七一的诗文，成一网站。未曾与叔父商量过“结庐”这一名字，只是觉得叔父具备中国传统文化中的“士”的气质，而陶渊明则是“士”之当之无愧的代表。希望叔父在酒泉之下能够同意。
